# SchaltKraft Website - Benutzerhandbuch

## Was ist dieses CMS?

Dieses Content Management System (CMS) ermöglicht Ihnen, die Inhalte Ihrer Website ohne Programmierkenntnisse zu bearbeiten.

---

## Wie funktioniert die Website?

```
Ihre Änderungen im CMS
        ↓
GitHub (Code-Speicher)
        ↓
Netlify (baut die Seite automatisch)
        ↓
www.schaltkraft.ch (Ihre Live-Website)
```

**Wichtige Adressen:**
- **CMS-Zugang:** www.schaltkraft.ch/keystatic
- **Website:** www.schaltkraft.ch
- **Domain-Anbieter:** Metanet

---

## Navigation im CMS

### Singletons (Einmalige Bereiche)
Diese Bereiche existieren nur einmal auf der gesamten Website:

| Bereich | Was Sie hier ändern können |
|---------|---------------------------|
| **Header / Kopfzeile** | Logo, Navigation, Menü-Links |
| **Footer / Fusszeile** | Kontaktdaten, Quick Links, Copyright |
| **SEO Einstellungen** | Standard-Titel und Beschreibung für Google |

### Collections (Sammlungen)
Hier können Sie mehrere Einträge erstellen:

| Sammlung | Beschreibung |
|----------|-------------|
| **Seiten** | Alle Unterseiten (Startseite, Über uns, AGB, etc.) |
| **Dienstleistungen** | Detaillierte Service-Seiten |
| **Team** | Mitarbeiter-Profile |
| **Jobs & Karriere** | Stellenausschreibungen |

## Seiten bearbeiten

### Seiten-Bausteine (Blocks)

Jede Seite besteht aus Bausteinen. Hier die verfügbaren:

| Baustein | Verwendung |
|----------|------------|
| **Grosser Startbereich** | Prominenter Bereich oben mit Haupttext und Button |
| **Vertrauensleiste** | 3 kurze Aussagen (z.B. "20 Jahre Erfahrung") |
| **Text mit Bild** | Text links, Bild rechts (oder umgekehrt) |
| **Partner & Zertifikate** | Animierte Logo-Leiste |
| **Leistungs-Kacheln** | Service-Übersicht mit Bildern |
| **Werte / Vorteile** | Karten mit Icons und Beschreibung |
| **Kundenstimmen** | Zitate von Kunden mit Sterne-Rating |
| **Kontakt-Formular** | Formular für Anfragen |
| **Stellenangebote** | Zeigt automatisch alle Jobs an |
| **Team-Übersicht** | Zeigt automatisch alle Teammitglieder |
| **Freier Textbereich** | Flexibler Text für AGB, Impressum, etc. |

## Inhalte aktualisieren

### Text ändern
1. Klicken Sie auf den Bereich
2. Ändern Sie den Text
3. Klicken Sie auf **Done** (Fertig)
4. Klicken Sie auf **Save** (Speichern)

### Bild ändern
1. Klicken Sie auf das Bild-Feld
2. Wählen Sie **Choose file** oder ziehen Sie ein Bild hinein
3. Speichern Sie

### Neuen Block hinzufügen
1. Klicken Sie auf **Add** (Hinzufügen)
2. Wählen Sie den gewünschten Baustein
3. Füllen Sie die Felder aus
4. Speichern Sie

## Links erstellen

### Interne Links
Geben Sie nur den Pfad ein, z.B.:
- `/kontakt`
- `/ueber-uns`
- `/dienstleistungen`

### Externe Links
Geben Sie die vollständige URL ein:
- `https://www.google.ch`
- `https://www.linkedin.com/company/schaltkraft`

Externe Links öffnen automatisch in einem neuen Tab.

---

## Bilder einfügen

### Im Texteditor (Freier Textbereich)
1. Klicken Sie auf das **+** Symbol
2. Wählen Sie **Bild einfügen**
3. Laden Sie das Bild hoch
4. Wählen Sie:
   - **Grösse:** Klein (25%), Mittel (50%), Gross (75%), Volle Breite
   - **Ausrichtung:** Links, Zentriert, Rechts

### Empfohlene Bildformate
- **Format:** JPG oder PNG
- **Grösse:** Max. 2MB pro Bild
- **Auflösung:** Min. 800px Breite

---

## Nach dem Speichern

Ihre Änderungen werden automatisch veröffentlicht:

1. Sie speichern im CMS
2. GitHub erhält die Änderung (automatisch)
3. Netlify baut die Seite neu (ca. 1-2 Minuten)
4. Die Website zeigt die neuen Inhalte

**Tipp:** Warten Sie ca. 2 Minuten nach dem Speichern, bevor Sie die Live-Seite prüfen.

## Häufige Fragen

### Wann werden meine Änderungen sichtbar?
Nach ca. 1-2 Minuten. Netlify baut die Seite automatisch neu.

### Kann ich etwas kaputt machen?
Nein. Alle Änderungen werden in GitHub gespeichert. Bei Problemen kann der Entwickler alte Versionen wiederherstellen.

### Warum sehe ich manche Änderungen nicht sofort?
Ihr Browser speichert die alte Version. Drücken Sie **Ctrl+F5** (Windows) oder **Cmd+Shift+R** (Mac) um neu zu laden.
